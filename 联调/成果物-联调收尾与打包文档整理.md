## 3. 项目打包说明

### 3.1 仓库交付结构（Monorepo）

```text
qt/
├── front/                 # A4 前端（Next.js 15）
├── backend/               # A5 数据服务 + API 文档 + data.sqlite3（联调库）
├── 联调/                  # 全链路脚本、A1/A2/A3 子模块、成果物
├── Readme.md              # 前后端联调总入口 ★
├── 课程工作报告_A4前端_Alpha组长.md
└── 范例.md
```

### 3.2 A4 前端打包

**环境**：Node.js 20.x，npm

```powershell
cd front
npm install
copy .env.local.example .env.local   # 或手动创建，见下
npm run typecheck                    # 类型检查
npm run lint                         # 可选
npm run build                        # 生产构建
npm start                            # 生产启动 :3000
```

**`.env.local` 必填项**：

```env
NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:8000
```

**开发模式（联调推荐）**：

```powershell
npm run dev          # Turbopack，:3000
```

**构建产物**：`front/.next/`（生产部署目录；`.gitignore` 不提交，本地 `npm run build` 生成）

### 3.3 A5 后端打包

**环境**：Python 3.10+，pip

```powershell
cd backend
pip install -r requirements.txt
# 可选：set APP_DB_PATH=...\data.sqlite3
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

**依赖**：`fastapi`、`uvicorn`（见 `backend/requirements.txt`）

**交付物**：
- 源码：`backend/app/`
- 联调数据库：`backend/data.sqlite3`（可选随库交付）
- 静态媒体：`backend/app/static/`（如 `en.wav`）
- 接口文档：`backend/API_数据库对接文档.md`

### 3.4 联调子模块（按需打包）

| 模块 | 目录 | 启动 | 端口 |
|------|------|------|------|
| A2 | `联调/ATC-VA-A2/` | `python run.py` | 8001 |
| A3 | `联调/a3_speech_processing_6/` | `uvicorn app.main:app --port 9002` | 9002 |
| A1 演示 | `联调/ATC-ADSB-Receiver/` | 静态页（数据经 sync 入 A5） | — |

**一键四服务 + 数据**：

```powershell
cd 联调
.\start-all.ps1
Start-Sleep -Seconds 15
.\health-check.ps1
python sync_all_to_a5.py
```

### 3.5 课程验收演示包（推荐最小集）

教师/组员 **仅验收标注闭环** 时，最小运行集为：
